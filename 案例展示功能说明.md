# 案例展示功能说明

## 📋 功能概述

新增案例展示页面，支持图片和视频的混合展示，使用Vant Swiper组件实现轮播循环播放，为用户提供直观的项目案例参考。

---

## ✨ 核心功能

### 1. 案例列表展示
- **分类筛选**：支持按服务类型筛选案例
  - 全部案例
  - 无人机物流
  - 政务巡检
  - 无人机托管
  - 无人机吊运
  - 航空表演
  - 影视航拍

- **案例卡片**：
  - 封面图片/视频预览
  - 案例标题和描述
  - 媒体类型标签（图片/视频）
  - 浏览量和发布时间

### 2. 案例详情
- **媒体轮播**：
  - ✅ 支持图片和视频混合展示
  - ✅ 自动循环播放（3秒间隔）
  - ✅ 手动滑动切换
  - ✅ 轮播指示器
  - ✅ 视频控制器（播放/暂停/进度条）

- **详细信息**：
  - 服务类型
  - 项目地点
  - 完成时间
  - 案例介绍
  - 项目亮点

### 3. 页面入口
- **首页入口**：精选案例横幅
- **个人中心**：案例展示菜单项
- **直接访问**：`/cases` 路由

## 🎬 媒体轮播功能

### Swiper配置
```javascript
<van-swipe
  :autoplay="3000"        // 自动播放，3秒切换
  :loop="true"            // 循环播放
  indicator-color="#667eea" // 指示器颜色
>
  <van-swipe-item>
    <!-- 图片或视频 -->
  </van-swipe-item>
</van-swipe>
```

### 支持的媒体类型
1. **图片**
   - 自适应容器
   - 保持宽高比
   - 支持点击查看

2. **视频**
   - HTML5 video标签
   - 自定义封面图
   - 播放控制器
   - 移动端兼容（playsinline）


## 🚀 使用说明

### 访问案例展示
1. **方式一**：首页点击"精彩案例集锦"横幅
2. **方式二**：个人中心点击"案例展示"菜单
3. **方式三**：直接访问 `/cases` 路由

### 查看案例详情
1. 在案例列表点击任意案例卡片
2. 弹出详情弹窗
3. 查看媒体轮播和详细信息
4. 点击关闭按钮或下滑关闭

### 筛选案例
1. 点击顶部分类Tab
2. 选择感兴趣的服务类型
3. 列表自动筛选对应案例
