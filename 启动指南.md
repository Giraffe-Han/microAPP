# 低空综合服务平台 - 快速启动指南

## 🚀 快速开始

### 1. 进入项目目录

```powershell
cd h5-platform
```

### 2. 安装依赖

```powershell
npm install
```

> 💡 首次安装可能需要几分钟时间

### 3. 启动开发服务器

```powershell
npm run dev
```

### 4. 访问应用

浏览器打开：**http://localhost:5173**

---

## 📱 手机端访问

### 方法一：局域网访问

1. 确保手机和电脑在同一WiFi
2. 查看电脑IP地址（PowerShell执行）：
   ```powershell
   ipconfig
   ```
   找到 "无线局域网适配器 WLAN" 下的 IPv4 地址

3. 手机浏览器访问：
   ```
   http://[你的IP]:5173
   ```
   例如：http://192.168.1.100:5173

### 方法二：浏览器模拟器

1. 在Chrome浏览器打开 http://localhost:5173
2. 按F12打开开发者工具
3. 点击设备工具栏图标（或按Ctrl+Shift+M）
4. 选择移动设备型号（如iPhone 12 Pro）

## 🔧 常见问题

### Q: 启动失败？
A: 
1. 确保已安装Node.js（版本>=16）
2. 删除node_modules文件夹，重新npm install
3. 检查5173端口是否被占用

### Q: 页面空白？
A: 
1. 打开浏览器控制台查看错误
2. 确认是否正确访问http://localhost:5173
3. 尝试清除浏览器缓存

### Q: 手机无法访问？
A: 
1. 确认手机和电脑在同一WiFi
2. 确认IP地址正确
3. 关闭电脑防火墙试试

### Q: 地图功能没有？
A: 地图功能目前显示"开发中"提示，需要后续集成高德地图SDK
