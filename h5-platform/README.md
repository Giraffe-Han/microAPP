# 低空综合服务平台 - H5移动端

## 项目介绍

低空综合服务平台H5移动端，参考[时尚科技官网](https://www.dj978.com/)设计，提供十四项低空服务的在线申请与管理功能。

## 技术栈

- **Vue 3** - 渐进式JavaScript框架  
- **Vite 5** - 下一代前端构建工具  
- **Vant 4** - 轻量、可靠的移动端组件库  
- **Vue Router 4** - 官方路由  
- **Pinia 2** - 状态管理  
- **Axios** - HTTP客户端

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

访问：http://localhost:5173

### 手机端调试

确保手机和电脑在同一局域网，访问：http://[你的IP]:5173

或使用Chrome开发工具的设备模拟器

## 页面路由

- `/home` - 服务大厅
- `/services` - 全部服务
- `/service-detail/:id` - 服务详情
- `/service-apply/:id` - 服务申请
- `/applications` - 我的申请
- `/mine` - 个人中心
- `/login` - 登录注册
