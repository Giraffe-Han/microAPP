## 🚀 快速启动

### 1. 启动服务器
```powershell
# 进入项目目录
cd "D:\低空\开发\低空综合服务平台\h5-platform"

# 启动开发服务器
npm run dev
```

**或使用启动脚本：**
双击 `h5-platform/启动服务.bat`

### 2. 访问物流服务申请表单
```
方法1：http://localhost:5173/ → 点击"无人机物流" → "立即申请"
方法2：http://localhost:5173/services → 点击"无人机物流" → "立即申请"
方法3：直接访问 http://localhost:5173/service-apply/1
```

## 🐛 问题排查

### 问题1：选择器不弹出
**可能原因：**
- JavaScript错误
- 组件未正确导入

**解决方法：**
1. 按F12打开开发者工具
2. 查看Console错误
3. 刷新页面重试

### 问题2：图片上传失败
**可能原因：**
- 文件格式不支持
- 文件过大

**解决方法：**
1. 确保上传的是图片格式（jpg/png/gif）
2. 检查文件大小
3. 查看Console错误信息

### 问题3：表单验证不通过
**可能原因：**
- 必填项未填写
- 格式不正确

**解决方法：**
1. 检查所有必填项（标红的字段）
2. 确认手机号格式正确（11位数字）
3. 查看具体错误提示

### 问题4：提交后没有反应
**可能原因：**
- JavaScript错误
- 网络问题

**解决方法：**
1. 查看Console错误
2. 检查Network请求
3. 刷新页面重试
