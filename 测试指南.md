## 🚀 快速启动

### 方法1：使用启动脚本（最简单）
1. 进入目录：`D:\低空\开发\低空综合服务平台\h5-platform`
2. 双击运行：`启动服务.bat`
3. 等待启动完成
4. 浏览器自动打开或手动访问：`http://localhost:5173`

### 方法2：命令行启动
```powershell
# 进入项目目录
cd "D:\低空\开发\低空综合服务平台\h5-platform"

# 启动开发服务器
npm run dev
```

### 方法3：直接启动（已在h5-platform目录）
```powershell
npm run dev
```

## 🐛 常见问题排查

### 问题1：服务器启动失败
**错误信息：**
```
npm error code ENOENT
```

**解决方法：**
```powershell
# 确保在正确的目录
cd "D:\低空\开发\低空综合服务平台\h5-platform"

# 检查package.json是否存在
dir package.json

# 如果不存在，说明目录错误
```

### 问题2：端口被占用
**错误信息：**
```
Port 5173 is in use
```

**解决方法：**
1. Vite会自动使用下一个端口（5174）
2. 查看终端显示的实际端口号
3. 使用新端口访问

### 问题3：页面空白
**可能原因：**
- JavaScript错误
- 路由配置问题

**解决方法：**
1. 按 `F12` 打开控制台
2. 查看Console错误信息
3. 检查Network请求状态

### 问题4：新服务不显示
**检查项：**
1. 确认文件已保存
2. 刷新浏览器（`Ctrl + F5` 强制刷新）
3. 检查控制台错误
4. 重启开发服务器
