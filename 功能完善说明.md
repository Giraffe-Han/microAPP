## 🚀 用户操作流程

### 场景1：申请重点服务（1-4号）
1. 用户首页点击"无人机物流"
2. 查看服务详情
3. 点击"立即申请"
4. 填写完整表单（货物信息、地点等）
5. 提交成功 → 跳转到"我的申请"

### 场景2：浏览其他服务（5-13号）
1. 用户点击"查看更多服务"
2. 浏览13项服务列表
3. 点击"航空摄影测量"
4. 查看服务详情（口号、项目、优势）
5. 点击"立即申请"
6. 看到"即将开放"提示
7. 可选择拨打客服电话或返回
