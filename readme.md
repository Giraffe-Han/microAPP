# 低空综合服务平台 - 一期工程

## 🎯 项目概述
低空综合服务平台旨在为低空经济提供综合性的服务对接平台，通过用户填报表单递交申请，后续人工联系用户进行业务对接的模式，快速实现平台上线运营。

## 💼 业务模式（一期）
- 用户通过平台填写服务申请表单
- 平台接收并记录用户需求
- 后台管理员查看申请信息
- 人工联系用户进行业务对接
- **订单管理、费用结算等功能计划安排在二期工程**

## 🛠️ 技术栈

- **Vue 3** - 使用Composition API
- **Vite 5** - 快速的开发服务器
- **Vant 4** - 移动端UI组件库
- **Vue Router 4** - 路由管理
- **Pinia 2** - 状态管理（已配置）
- **Axios** - HTTP请求（已配置）
