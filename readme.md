
# 畅行温州-云享飞 微信小程序

## 项目概述

**畅行温州-云享飞** 是由交运集团低空发展有限公司开发的综合性无人机服务平台微信小程序。该平台致力于为用户提供全方位的无人机相关服务，包括维修、托管、表演、租赁、培训等多元化业务。

### 参考项目
- 新翼飞布局及功能参考：[新浪财经报道](https://finance.sina.com.cn/roll/2025-09-15/doc-infqpxhk9798768.shtml)

## 技术栈

- **前端框架**: 微信小程序原生开发
- **界面技术**: H5界面实现
- **后端服务**: 服务器部署支持
- **数据管理**: 订单管理、进度跟踪、费用结算

## 功能模块

### 核心业务模块

#### 1. 无人机维修服务
- **服务范围**: 大疆过保后维修服务
- **核心功能**:
  - 维修订单管理
  - 维修进度实时跟踪
  - 维修费用结算
  - 维修报告生成

#### 2. 无人机托管服务
- **服务内容**: 专业无人机托管管理
- **核心功能**:
  - 托管订单管理
  - 托管进度跟踪
  - 托管费用结算
  - 设备状态监控

#### 3. 无人机表演服务
- **服务特色**: 专业无人机编队表演
- **核心功能**:
  - 表演订单管理

  - 设备使用监控

#### 5. 城市治理服务

  - 治理进度跟踪
  - 治理费用结算
  - 治理效果评估

#### 6. 飞手培训服务
- **培训内容**: 专业无人机操作培训
- **核心功能**:

## 开发环境配置

### 前置要求
- 微信开发者工具
- Node.js (推荐 v16+)
- 微信小程序账号

### 安装步骤
1. 克隆项目到本地
2. 使用微信开发者工具打开项目
3. 配置小程序 AppID
4. 安装依赖包
5. 启动开发服务器

## 部署指南

### 开发环境部署
1. 配置开发环境参数
2. 启动本地开发服务器
3. 在微信开发者工具中预览

### 生产环境部署
1. 构建生产版本
2. 上传代码到微信小程序后台
3. 提交审核
4. 发布上线
