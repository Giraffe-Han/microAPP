## 📦 交付内容

### 项目文件
```
h5-platform/
├── src/
│   ├── views/           # 页面组件（7个）

│   │   └── auth/        # 登录注册
│   ├── router/          # 路由配置
│   ├── styles/          # 全局样式
│   ├── App.vue          # 根组件
│   └── main.js          # 入口文件
├── package.json         # 项目依赖
├── vite.config.js       # Vite配置
└── README.md            # 项目说明
```

### 文档文件
- `readme.md` - 项目总体说明

## 🚀 如何运行

### 1. 安装依赖
```powershell
cd h5-platform
npm install
```

### 2. 启动项目
```powershell
npm run dev
```

### 3. 浏览器访问
**电脑：** http://localhost:5173  
**手机：** http://[你的IP]:5173

### 4. 体验功能
1. 首页查看服务
2. 点击服务查看详情
3. 填写申请表单
4. 查看我的申请
5. 进入个人中心

## 🎯 页面路由

| 路径 | 页面 | 说明 |
|------|------|------|
| `/home` | 服务大厅 | 首页，展示服务入口 |
| `/services` | 全部服务 | 14项服务列表 |
| `/service-detail/:id` | 服务详情 | 服务介绍和说明 |
| `/service-apply/:id` | 服务申请 | 填写申请表单 |
| `/applications` | 我的申请 | 查看申请记录 |
| `/mine` | 个人中心 | 用户信息和设置 |
| `/login` | 登录注册 | 短信验证码登录 |

## 💡 技术说明

### 技术栈
- **Vue 3.4** - Composition API
- **Vite 5.2** - 快速构建工具
- **Vant 4.8** - 移动端UI组件库
- **Vue Router 4.3** - 路由管理
- **Pinia 2.1** - 状态管理（已配置）
- **Axios 1.6** - HTTP请求（已配置）

### 数据说明
- 当前所有数据都是前端模拟数据
- 用于界面展示和交互演示
- 等待后端API接口开发后对接真实数据

### 接口预留
- 用户登录/注册接口
- 服务列表查询接口
